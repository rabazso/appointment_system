## Booking Page
- **Appointment Process:**
    - This is the core function of the application where logged-in users can reserve a time slot.
- **Selection Steps:**
    1. **Service Selection:** Users choose the type of service they need (e.g., Haircut, Beard Trim, Shave).
    2. **Barber Selection:** If not already selected, the user picks a barber.
    3. **Date & Time:** A calendar interface allows users to pick a date.
- **Validation & Rules:**
    - **No Past Dates:** The system prevents selecting dates in the past.
    - **Opening Hours:** Users can only select times within the shop's opening hours.
    - **Availability:** Slots that are already booked by other customers are disabled to prevent double-booking.
- **Confirmation:**
    - Upon successful submission, the appointment is saved to the user's profile.
